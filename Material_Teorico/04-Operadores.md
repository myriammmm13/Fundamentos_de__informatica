# ¿Qué son los operadores?
Son operadores, símbolos o palabras clave que funcionan con un significado específico, haciendo esto que se puedan realizar distintas operaciones con una p más variables para producir algún tipo de resultado.

# ¿Para qué sirven los operadores?
- Realizar cálculos matemáticos
- Comparar valores
- Evaluar condiciones
- Asignar datos a variables
- Usar lógica booleana

# Tipos de operadores
Existen distintos operadores, mismos que según su tipo tendrán una forma de funcionar distinta, por ejemplo:

## Aritméticos
Estos, como su nombre lo dice, funcionan para realizar funciones aritméticas.
- Suma: +
- Resta o decremento: -
- Multiplicación: *
- División: /
- Módulo o residuo: %

## Asignación
Estos funcionan para poder asignar distintos valores a las variables.
- Asignar: =
- Valor anterior más: += o variable = variable + valor
- Valor anterior menos: -= o variable = variable - valor
- Valor anterior por: *= o variable = variable * valor
- Valor anterior dividido: /= o variable = variable / valor

## Comparación
Estos se utilizan dentro de condicionales, iterativas, operadores ternarios y otros. Se deben utilizar con dos variables o más y retornan de forma booleana (true o false).
- Igual a: ==
- Mayor: >
- Menor: <
- Mayor o igual: >=
- Menor o igual: <=
- Diferente a: !=

## Lógicos 
Se utilizan para poder trabajar la álgebra booleana, combinando así distintas condiciones.
- AND(y): &&
- OR (o): ||
- NOT (no o negación): !

## Incremento o degremento
Estos son utilizados para hacer incrementos o decredementos de forma unitaria (para hacerlo de uno en uno) para ocasiones necesarias, como en las iterativas. Este tipo de operador es utilizado en C++ (lenguaje que se aprenderá en fundamentos de informática).
- Incremento: ++
- Decremento: --

Un dato importante de estos operadores es que su función se hará de distinta forma según su posición con respecto al nombre de la variable.
Si la escritura está en pre-incremento "++variable" esta se hará en la misma línea donde se llame a la acción, mientras que si está en post-incremento "variable++" se realizará el aumento hasta la siguiente línea de donde se llame a realizar la acción. Esto aplica de igual manera cuando son decrementos.
