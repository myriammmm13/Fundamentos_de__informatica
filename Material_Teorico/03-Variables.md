## ¿Qué son las variables en la programación?
Las variables son, en términos básicos, un espacio o almacenamiento donde se pueden guardar diferentes tipos de datos o información. 
Existen distintos tipos de "cajitas" o almacenamientos: 
# Primitivos
-Numéricos (enteros o decimales)
-Booleanos (verdadero o falso, 1 o 0)
-Caractér ('a' o '"#')

# No primitivos o de referencia
-Cadena de texto o String ("Hola Mundo!")
-Objetos (esto lo verás más adelante en POO)
-Arreglos (se guardan múltiples variables de un mismo tipo, ej: [1, 2, 4, 34, 7])
