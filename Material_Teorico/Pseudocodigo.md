# ¿Qué es el pseudocódigo?
El pseudocódigo es una herramienta utilizada en la programación para poder comenzar a desarrollar el pensamiento lógico necesario para la resolución de promblemas. Este es una forma "máquina" de escribir los pasos a realizar para resolver un problema en especíico o para realizar un proceso necesario.

# ¿Para qué sirve el pseudocódigo?
Este es sumamente útil para desarrollar la lógica de programación, ya que no necesitas aprender la sintaxis de un lenguaje de programación para poder escribir las instrucciones de la tarea a realizar, haciendo esto que no sea un proceso tan complejo o tedioso si aún no sabes programar ni un lenguaje de programación. Además, con este puedes practicar el desarrollo o resolución más ágil de problemas de programación. 

Como punto a resaltar, el pseudocódigo se puede tomar como la base para poder aprender a programar en cualquier lenguaje de programación, ya que todos llevan una funcionalidad similar por dentro, únicamente cammbia la sintaxis y semántica de estos. Por otro lado, este te permite centrarte completamente en la lógica, ya que no debes preocuparte por los errores de sintaxis ni en su ejecución.

# ¿Cómo puedo comenzar a utilizar el pseudocódigo para prácticar?
Muchas veces puedes realizar este a papel o en un bloc de notas sin problema alguno, ya que su punto importante es practicar; sin embargo, si quieres ir verificando la funcionalidad de lo que realices, puedes utilizar el programa Pseint para practicar.
