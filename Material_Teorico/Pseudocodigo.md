# ¿Qué es el pseudocódigo?
El pseudocódigo es una herramienta utilizada en la programación para poder comenzar a desarrollar el pensamiento lógico necesario para la resolución de promblemas. Este es una forma "máquina" de escribir los pasos a realizar para resolver un problema en especíico o para realizar un proceso necesario.

# ¿Para qué sirve el pseudocódigo?
Este es sumamente útil para desarrollar la lógica de programación, ya que no necesitas aprender la sintaxis de un lenguaje de programación para poder escrbiri las instrucciones de la tarea a realizar, haciendo esto que no sea un proceso tan complejo o tedioso si aún no sabes programar ni un lenguaje de programación.
